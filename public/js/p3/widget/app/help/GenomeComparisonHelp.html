Genome Selection and Parameters

Select a reference genome
Type or select a genome name as a reference genome.

Select up to 8 genomes to compare
Type or select a genome name and use the plus button to place the genome to the table

Optional External Genome (Protein Fasta file)
Select or upload an external genome file in protein fasta format. This field is optional.

Advanced parameters:

Minimum % coverage
Minimum percent sequence coverage of query and subject in blast. Use up or down arrows to change the value. The default value is 30%
Blast e-value
Maximum blast e-value. A default value of 1e-5 is used if leave blank.

Output Folder
The workspace folder where results will be placed.

Output Name
Name used to uniquely identify results.
