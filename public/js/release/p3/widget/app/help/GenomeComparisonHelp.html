Genome Selection and Parameters

Select a reference genome
Type or select an genome name as a reference genome.

Select up to 8 genomes to compare
Type or select an genome name and use the plus button to place the genome to the table

Optional user genome
Select or upload an user genome file in protein fasta format. This field is optional.

Advanced parameters:

Blast hit percent coverage
Minimum percent coverage of the blast hits. Use up or down arrows to change the value.
Blast e-value
Maximum blast e-value. A default value of 1e-5 is used if leave blank.

Output Folder
The workspace folder where results will be placed.

Output Name
Name used to uniquely identify results.
